# 🧩 Sudoku Game — 数独挑战

一个使用 **Python 3.13** 编写的终端交互式数独游戏，支持随机生成关卡、多档难度选择、智能提示和即时验证，适合在命令行中轻松享受数独乐趣。

---

## ✨ 功能特色

- 🎲 **随机生成关卡**  
  每次开始游戏都会生成一个合法且唯一解的数独谜题，保证可解。
- 📊 **四种难度**  
  可选 **简单 / 中等 / 困难 / 专家**，难度由初始已知格数量控制（从 ~40 格到 ~22 格）。
- 💡 **智能提示**  
  卡关时可获取当前格子的正确数字（基于求解算法），帮助学习或推进。
- ✅ **即时校验**  
  每步填写都会检查是否与同行、同列、同宫冲突，并给出错误反馈。
- ⌨️ **纯键盘交互**  
  使用数字键 `1-9` 填写，方向键或 `WASD` 移动光标，`H` 获取提示，`N` 重新开局，`Q` 退出。
- 🎨 **彩色终端界面**  
  使用 ANSI 颜色区分初始数字、用户填写数字、错误高亮，视觉清晰。

---

## 🚀 快速开始

### 环境要求
- Python 3.13 或更高版本（推荐使用 3.13）
- 终端支持 ANSI 颜色（Windows 10+ / macOS / Linux 默认支持）
